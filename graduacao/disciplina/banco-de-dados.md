## Ementa

Modelo Entidade-Relacionamento. Tipos de entidades, conjuntos de
entidades e atributos-chaves. Tipos de relacionamentos, papéis e

NoSQL. Bancos de Dados RDF. Manipulação de bases de dados publicamente
disponíveis.
