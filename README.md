
# Site EMAp

This is the source of EMAp website.

## Deploy

- git checkout source
- do whatever you need
- git status / add / commit
- jekyll
- checkout master
- execute 

    cp -r _site/* . && rm -rf _site/ && touch .nojekyll

- git status
- git add / commit
- git push -all origin
